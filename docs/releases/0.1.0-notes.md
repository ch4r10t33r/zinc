# ZINC v0.1.0

ZINC v0.1.0 is the first binary release of the ZINC inference engine.

## Downloads

- Linux x86_64: `zinc-v0.1.0-linux-x86_64.tar.gz`
- macOS Apple Silicon: `zinc-v0.1.0-macos-aarch64.tar.gz`
- Checksums: `SHA256SUMS.txt`

## Install

Linux:

```bash
tar -xzf zinc-v0.1.0-linux-x86_64.tar.gz
cd zinc-v0.1.0-linux-x86_64
./bin/zinc --version
./bin/zinc --check
```

macOS:

```bash
tar -xzf zinc-v0.1.0-macos-aarch64.tar.gz
cd zinc-v0.1.0-macos-aarch64
./bin/zinc --version
./bin/zinc --check
```

If the macOS archive is not notarized, users may need:

```bash
xattr -dr com.apple.quarantine .
```

## Quick Start

```bash
./bin/zinc model list
./bin/zinc model pull <model-id>
./bin/zinc --model-id <model-id> --prompt "Hello" --chat
```

## Server Mode

```bash
./bin/zinc --model-id <model-id> --port 9090
curl http://127.0.0.1:9090/health
```

## Supported Platforms

- Linux x86_64 with a Vulkan-capable AMD GPU.
- Linux x86_64 with an Intel Arc Xe2/Battlemage GPU.
- macOS Apple Silicon with Metal.

## Supported Model Families

- Qwen3 / Qwen3.5
- Gemma 4

## Known Limitations

- Early release.
- No Windows binary.
- No bundled models.
- macOS binary may be unsigned or not notarized.
- Server generation is serialized.
- Intel Arc is supported through the Linux Vulkan build, but Arc-specific tuning is younger than the RDNA4 path.
